# Base Tokens

The starter base tokens live in `@lmfaole/basics/basic-styling/tokens/base.css`.
They cover typography, spacing, and shape — everything the components consume that is not a palette color.
Override them on `:root` (or any subtree) to retune the visual scale without rewriting component CSS.

## Spacing

The spacing scale is a five-step progression. Compositional tokens (`--basic-flow-space`, `--basic-panel-padding`, `--basic-control-padding-*`) resolve to specific steps so components keep a consistent rhythm.
