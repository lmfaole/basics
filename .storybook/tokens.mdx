# Base Tokens

The starter base tokens live in `@lmfaole/basics/basic-styling/tokens/base.css`.
They cover typography (fluid by default), spacing, and shape — everything the components consume that is not a palette color.
Override them on `:root` (or any subtree) to retune the visual scale without rewriting component CSS.

## Fluid Typography

The font-size tokens are fluid — they scale between a minimum and maximum across a viewport range. Each size resolves to:

```
clamp(--basic-font-size-*-min, lerp, --basic-font-size-*-max)
```

…where the middle term linearly interpolates as the viewport grows from `--basic-fluid-min-viewport` to `--basic-fluid-max-viewport`.

A single multiplier — `--basic-fluid-scale` — controls how aggressively the type scales. Override it on `:root` or any subtree:

```css
:root {
    --basic-fluid-scale: 0;     /* disable fluidity (every size pins to its min) */
    --basic-fluid-scale: 0.5;   /* gentler scaling, reaches max past the configured viewport */
    --basic-fluid-scale: 1;     /* default — reaches max at --basic-fluid-max-viewport */
    --basic-fluid-scale: 2;     /* bolder — reaches max at half the configured range */
}
```

Retune the viewport range with `--basic-fluid-min-viewport` (default `20rem` / 320px) and `--basic-fluid-max-viewport` (default `80rem` / 1280px). Override individual size bounds (e.g. `--basic-font-size-title-max`) when one step needs different limits.
