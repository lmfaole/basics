# Palette Tokens

The starter palette tokens are exported from `@lmfaole/basics/basic-styling/tokens/palette.tokens.json`.
They follow the W3C Design Tokens Community Group format with `$schema`, nested groups, `$type`, `$description`, and `$value`.

{paletteDocs.map((palette) => (
  <div key={palette.id}>
    <h2>{palette.title}</h2>
    <p>{palette.description}</p>

    <h3>Base Tokens</h3>
    <ColorPalette>
      {palette.baseRows.map((row) => (
        <ColorItem key={row.id} title={row.title} subtitle={row.subtitle} colors={row.colors} />
      ))}
    </ColorPalette>

    <h3>Computed Tokens</h3>
    <ColorPalette>
      {palette.computedRows.map((row) => (
        <ColorItem key={row.id} title={row.title} subtitle={row.subtitle} colors={row.colors} />
      ))}
    </ColorPalette>
  </div>
))}
